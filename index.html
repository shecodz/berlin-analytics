Project Information

This is a Node.js/Express e-commerce platform with EJS templating.

To run locally:

  1. Clone this repository
  2. Run npm install
  3. Run node src/server.js
  4. Visit http://localhost:3000

GitHub Pages cannot host the full functionality of this application. Please run it locally for the complete experience.
